# Personal Profile Card
## Description
A static profile card that displays a user's avatar, name, bio,
skill tags, and contact action buttons.
The layout uses Cards with elevation, Columns and Rows for structure,
and Icons for visual cues.
A focused exercise in composing polished, real-world layouts with Material Design 3 components.
## Screenshots

## Tech Stack
- Language: Kotlin
- UI: Jetpack Compose + Material Design 3
- Image Loading: Coil (bonus)
## Features
- Rounded profile card with avatar, name, and bio
- Horizontally scrollable skill tags
- Contact action buttons (Call, Email, Share)
- Dark mode toggle (bonus)
## Setup
1. Clone the project.
2. Open in Android Studio and sync.
3. Run on API 26+ device or emulator.
## Architecture
Single-activity, single-composable layout with data-driven design.
